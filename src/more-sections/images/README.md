# 图片文件夹

此文件夹用于存放 Markdown 文件中引用的图片。

## 使用方法

在 Markdown 文件中使用以下语法引用图片：

```markdown
![图片描述](images/图片文件名.png)
```

或者使用相对路径：

```markdown
![图片描述](./images/图片文件名.png)
```

## 支持的图片格式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- SVG (.svg)
- WebP (.webp)

## 注意事项

- 图片文件会被自动复制到 `dist/more-sections/images/` 目录
- 建议使用相对路径引用图片
- 图片会自动适应容器宽度，保持宽高比
